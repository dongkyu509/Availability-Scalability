<H1>Sharding</H1>

오라클 데이터베이스 12c R2 이후의 새로운 기능으로, 기존 Shared Disk 방식의 RAC와 다르게 물리적으로 데이터를 분리하여 저장하는 수평적 파티셔닝 형태의 Shard를 지원합니다. 이를 통해서 수평적인 Scale-Out 의 확장성과 함께 ADG, RAC 등 기존의 고가용성 구조를 통한 분산된 가용성을 지원합니다.
